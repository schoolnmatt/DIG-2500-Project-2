# DIG-2500-Project-2, Community Food Bank Resource Page

## Project Description
The interface is intended to be an informational directory, allowing users to find
events, resources, and socials to connect them with their local food bank.

## Intended User or Player
Intended users are community members in need of help obtaining food security and
volunteers who want to support the goals of the food bank.

## User / Player Goal
Users’ primary goal is to learn when and where they are able to obtain food.
Secondary goals include connecting with health and wellness resources,
finding out about community events, and learning about volunteer opportunities.

## Inspiration Interfaces
- KnightConnect - Idea to group links together, and to establish hierarchy starting at top of page
- Orange County Library System - Upcoming events section provided inspiration
- Feeding America - Idea for changing color of buttons on hover
- IRC - Helped to inform use of color to highlight information
- WWF - News section adapted to build trust and credibility

## Design Choices
- Layout: Elements on the page are grouped together into grids depending on related content. These grids are placed on the page following visual and informational hierarchy. 
- Color: Green, black, white, and yellow are used together to elicit a natural and calm mood.
- Typography: Main heading twice the size of next smallest, then 10px less per subsequent heading
- Visual hierarchy: Most important elements placed at top of page, least important elements placed at the bottom. Images are given the most attention, directing the eye to associated text.
- Feedback states: Text changes color on hover, pictures translate upwards on hover.
- Responsive design: Grids dynamically resize depending on size of page, media query used to remove largest photo if the page is too small to properly display it.
